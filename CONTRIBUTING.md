# Contributing to Toolbox

Thanks for your interest. Toolbox is a library of small, self-contained Claude Code skills, commands, and scripts. Contributions that fit that shape are welcome.

## Before a big change

Open an issue (use the Proposal template) before starting significant work, so we can agree on the approach. Small fixes and docs can go straight to a pull request.

## The bar

- **Self-contained.** A skill or command brings everything it needs; a reader installs it by copying one folder or file.
- **Portable.** No absolute machine paths, no hardcoded personal directories, no references to private projects. It works for anyone who installs it.
- **Documented.** Add it to the README contents with a one-line description.

## Workflow

1. Fork and branch from `main`.
2. Add or change the skill, command, or script.
3. Open a pull request describing what and why, linked to its issue.
4. A maintainer reviews. Once approved, it merges.

## Style

Formatting is enforced by `.editorconfig` (tabs, K&R braces in any code). Keep comments minimal and only on non-obvious lines.
